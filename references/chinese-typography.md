# 中文排版与配色规范（Chinese Typography）

> 来源：W3C 中文排版需求（clreq）、Ant Design 字体/色彩规范、Apple HIG 中文版、
> 中文文案排版指北，2026-07 联网调研整理。中文排版不是英文规则的直译——
> 凡界面含中文，本文件优先级高于通用字体规则。

## 一、字体加载纪律（第一硬规则）

**装饰性中文 webfont 严禁滥用。** 中文字体单文件 5-20 MB，用户机器大概率没装
ZCOOL 系列这类字体，走 CDN 会严重拖慢首屏。因此：

- **默认：中文一律用系统字体栈**（零体积、渲染最佳）：

```css
font-family: -apple-system, BlinkMacSystemFont, "PingFang SC",
  "Hiragino Sans GB", "Microsoft YaHei", "Noto Sans SC", sans-serif;
```

- **中文界面的"个性"不靠堆中文 webfont**，靠这四样实现：
  西文/数字展示字体（体积小可随意换）、中文字重与字号对比、排版结构、色彩
- **例外（仅此两种场景可用装饰中文字体）**：
  1. 创意展示页（活动页/品牌 Hero/海报式页面）的**标题短语**
  2. 用户明确要求的书法/复古/手写气质
  且必须同时满足：只用于标题（正文/UI 永远系统栈）+ Google Fonts `text=` 参数
  子集化（只加载标题用到的字符）+ `font-display: swap` + 系统栈 fallback：

```html
<link href="https://fonts.googleapis.com/css2?family=Ma+Shan+Zheng&text=每场会议都值得铭记&display=swap" rel="stylesheet">
```

- 思源黑体/宋体（Noto Sans/Serif SC）全量引入也算违规——同样是几 MB 级别，
  标题需要时同样走子集化
- 风格试衣间的 mockup 属创意展示场景，可用装饰字体；但 Phase 3 交付的正式
  页面必须回到本纪律

**可选装饰中文字体清单（限上述例外场景）**：
Noto Serif SC（典雅书卷）· ZCOOL XiaoWei（轻盈时尚）· ZCOOL QingKe HuangYou
（圆润温暖）· ZCOOL KuaiLe（活泼跳动）· Ma Shan Zheng（毛笔楷书）·
Long Cang（随性手写）· Zhi Mang Xing（洒脱行书）

## 二、字重陷阱

- 多数中文字体**只有 400 / 700**，中间字重是引擎伪合成，渲染发虚
- 苹方有 Ultralight-Semibold **但没有 Bold**：`font-weight: 700` 实际落到 600
- 规范：中文正文 400，标题/强调写 **500 或 600**（不写 `bold`）；
  层级主要靠字号 + 颜色，字重只做辅助

## 三、排版参数

| 项 | 值 | 说明 |
|----|----|----|
| 正文字号 | Web ≥ 14px（阅读密度高用 16px）；App 正文 17pt | Ant Design / Apple HIG |
| 行高 | **1.5-1.75**（比英文的 1.4-1.6 更大） | 中文笔画密，需要更多呼吸 |
| 字距 | 正文 `letter-spacing: 0`；大标题可 +0.05em；**禁负字距** | 中文挤压会糊 |
| 每行字数 | 桌面 30-40 字，移动 18-25 字 | `max-width` 按此推算（约 `36em`） |
| 段落 | `margin-bottom: 1em`；不依赖 text-indent 首行缩进 | |

**盘古之白（中西文混排空格）**：
- 中文与英文之间加空格：`在 Lumina 上使用 SDK`
- 中文与数字之间加空格：`共 500 条记录`
- 数字与单位之间加空格：`20 TB`；**例外**：`90°`、`15%` 不加
- 全角标点前后不加空格

**标点规则**：
- 中文句子用全角标点（，。？！），不夹半角
- 破折号 `——` 占两字宽不拆行；省略号 `……` 同理
- 标点不叠用（禁「！！！」）
- **中文禁斜体**（无斜体字形设计，强制倾斜=笔画变形）；下划线只留给链接

## 四、中文界面配色

- 中性文字色用透明度写法适配多背景（Ant Design 体系）：
  亮色页 `rgba(0,0,0,.88)` 主文 / `.65` 次文 / `.25` 禁用；边框 `#d9d9d9`
- **暗色页正文不用纯白**：`rgba(255,255,255,.85)`，降低眩光
- 中文笔画在低对比暗底上损失更大：暗色正文对比度按 **7:1（AAA）** 设防，
  不是英文页常用的 4.5:1；品牌彩色不直接做暗底正文色
- 中国传统色（朱砂 `#C0392B`、松花绿 `#3B7A57`、靛青 `#1A3A4A`、琉璃黄
  `#F0C040`…）适合做点缀与营销页氛围，界面中性色骨架不变

## 五、数字与单位

- 数字一律半角 ASCII，禁全角（１２３）
- 界面数字（价格/统计/表格）：`font-variant-numeric: tabular-nums`
- 大数字展示配西文等宽/表格数字字体（体积小，可放心引入）：

```css
.metric { font-family: "Roboto Mono", "SF Pro Display", monospace;
          font-variant-numeric: tabular-nums; }
```

## 六、中文页 preflight 追加项

- [ ] 中文正文/UI 是系统字体栈；装饰中文字体只出现在标题且已子集化
- [ ] 行高 ≥ 1.5；正文 ≥ 14px；无负字距
- [ ] 中西文之间、中文与数字之间有空格；数字与单位间有空格（°、% 除外）
- [ ] 全角标点、无斜体、无叠用标点
- [ ] 中文字重只用 400/500/600，无伪 bold 发虚
- [ ] 暗色页正文 `rgba(255,255,255,.85)`，对比度 ≥ 7:1
